# Theseus, importing made easy

## Usage:
to use this, make sure you have:
1. A .env file with the required variables set, see `.env.example`
2. The required python packages installed.
3. The required changes in the importer.py file
4. A file you wish to import in the format of `trunc.csv`

Then run the following python command `python3 importer.py <your_file>`

NB: I wrote this library primarily for my own use case, however, suggestions are always welcome!
